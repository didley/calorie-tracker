## Setup
---

### Styling
Styling is provided by Bootsrap 4 CDN located within /client/public/index.html



### Installing
/server and /client directories have their own separate package.json files
```shell
$ cd /server
$ npm i
$ cd ..
$ cd /client
$ npm i
$ cd /client-2-mock-data
$ npm i
```

### Running
```shell
$ cd /server
$ npm start
$ cd ..
$ cd /client
$ npm start
$ cd /client-2-mock-data
$ npm start
```
